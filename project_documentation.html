1. System Overview

Keyhost is a comprehensive multi-vendor booking platform designed to facilitate seamless reservations for properties (stays) and flights. It serves three primary user roles: Guests (travelers), Property Owners (hosts), and Administrators.

Core Modules

  • Property Booking: Search, filter, and book diverse property types (Apartments, Hotels, Villas).
  • Flight Booking: Real-time flight search and booking integration (One-way, Round-trip, Multi-city).
  • User Management: Role-based access control and dashboard interfaces.
  • Financials: Earnings tracking, payouts, agency balance, and reward points system.

2. User Roles & Capabilities

Guest (Traveler) Guest

End-users looking to book services.

  • Search: Advanced search filters for location, dates, guests, and price range.
  • Booking: Secure booking flow for properties and flights.
  • Dashboard: View upcoming/past bookings, manage profile.
  • Rewards: Earn and redeem points for discounts.
  • Favorites: Save properties to a wishlist.
  • Messaging: Communicate directly with hosts.

Property Owner (Host) Owner

Partners listing their properties on the platform.

  • Property Management: Add new listings, edit details, upload photos, set pricing.
  • Booking Management: Accept/reject requests, view calendar availability.
  • Analytics: View performance metrics (views, bookings, revenue).
  • Earnings: Track payout history and pending balances.

Administrator Admin

Platform managers overseeing operations.

  • System Oversight: Manage all users, properties, and bookings.
  • Content Management: Configure amenities, property types, and site settings.
  • Financial Control: Process payouts, view platform earnings, manage reward points system.
  • Reviews: Moderation of user reviews and ratings.

3. Detailed Feature Manual

3.1 Search Functionality

The search system is the core entry point for users.

  • Global Search Bar Located in the Navbar and Sticky Header (on scroll). Allows quick access to location, dates, and guest selection.
  • Filters
    • Property Type: Tabs for Stays (Room, Apartment, Hotel) vs Flights.
    • Location: Autocomplete suggestions for cities and airports (with IATA codes).
    • Dates: Date range picker with support for check-in/check-out and flight departure/return.
    • Guests: Detailed counters for Adults, Children, Infants, and Pets.
  • Flight Specifics
    • Trip Type: Toggle between One Way, Round Trip, and Multi-city.
    • Class: Economy, Business, First Class selection.
    • Passenger Breakdown: Specific age-based passenger validation (e.g., Infants < 2 years).
    • Agency Balance: Wallet system for instant ticketing (B2B/Admin context).
  • Mobile Optimization
    • Sticky Search Header: Collapses to a pill view on scroll for maximum screen real estate.
    • Mobile Tabs: Swipeable property type tabs.
    • Mobile Filters: dedicated filter modal for easy access on small screens.

3.2 Booking Process

Property Booking

  1. Selection: User views property details (amenities, photos, rules).
  2. Request: User selects dates and guests to check availability.
  3. Confirmation: User proceeds to checkout (Guest details, Payment).

Flight Booking

  1. Search Results: List of available flights with airline logos, timings, stops, and refund status.
  2. Flight Details: Expandable view showing leg details, aircraft type, and baggage allowance.
  3. Fare Summary: Breakdown of base fare, taxes, and total per passenger.
  4. Passenger Details: Strict form validation for passport expiry and age compliance.
  5. Ticket Issuance:
    • Automated redirection to ticket success page upon booking.
    • Issue Ticket Page: For authorized users, a dedicated interface to confirm passengers and deduct cost from Agency Balance to issue final tickets (e-ticket generation).

5. User Guide (Walkthroughs)

5.1 Guest Guide

How to Book a Property

  1. Search: On the homepage, select the "Stays" tab. Enter your destination, check-in/out dates, and number of guests. Click Search.
  2. Browse & Filter: View the list of available properties. Use filters (Price, Review Score, Property Type) to narrow down choices.
  3. View Details: Click on a property card to view full details including description, amenities, host profile, and location map.
  4. Select Room/Unit: If applicable, choose the specific room or unit type.
  5. Reserve: Click Reserve or Request to Book.
  6. Checkout:
    • Review booking summary (dates, guests, total price).
    • Enter guest details.
    • Proceed to payment (Credit Card / Digital Wallet).
  7. Confirmation: You will receive a booking confirmation email and can view the trip in your Dashboard > My Trips.

How to Book a Flight

  1. Search: Navigate to the Flight tab. Select Trip Type (One Way / Round Trip / Multi-city).
  2. Enter Route: Input Origin (From) and Destination (To) airports. Select dates and travelers. Click Search.
  3. Compare: Review flight options. Expand details to see layovers, baggage allowance, and aircraft info.
  4. Select: Choose your preferred outbound (and return) flight. Click Book Now.
  5. Passenger Info: Enter details for all passengers exactly as they appear on passports. (Watch out for age-based validation for children/infants).
  6. Extras: Select add-ons like extra baggage or meals if available.
  7. Payment: Review the Fare Summary (Base Fare + Taxes) and complete payment.
  8. Ticket Issue: Receive your e-ticket immediately on the confirmation page and via email.

5.2 Property Owner (Host) Guide

Adding a New Property

  1. Login: Access your Owner Dashboard.
  2. Start Listing: Click Add New Property.
  3. Step 1 - Basic Info: Enter property title, description, and select category (e.g., Apartment, Villa).
  4. Step 2 - Location: Provide address and pin the exact location on the map.
  5. Step 3 - Details: Set guest capacity, number of bedrooms/bathrooms, and size (sq ft).
  6. Step 4 - Amenities: Check all available features (WiFi, Pool, AC, etc.).
  7. Step 5 - Photos: Upload valid, high-resolution images. Set a cover photo.
  8. Step 6 - Pricing: Set your nightly rate. Configure weekend or seasonal pricing if needed.
  9. Publish: Submit for admin review. Once approved, it goes live.

Managing Earnings & Payouts

  1. Track Revenue: Go to Dashboard > Earnings. View a graph of your income over time.
  2. Wallet Balance: Check your "Current Balance". This is the amount available for withdrawal (Total Earnings - Commission).
  3. Request Payout: Click Withdraw. Enter the amount and select your payout method (Bank Transfer / PayPal).
  4. History: View status of past withdrawal requests in the Payout History table.
